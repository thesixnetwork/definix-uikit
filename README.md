# Definix UIkit V2

## Install

`yarn add definixswap-uikit-v2`

## Setup

### Reset

A reset CSS is available as a global styled component.

```
import { ResetCSS } from 'definixswap-uikit-v2'
...
<ResetCSS />
```

### Types

This project is built with Typescript and export all the relevant types.
